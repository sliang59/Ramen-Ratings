What Should I Eat: A Look at Ramen Based on their Reviews

I am going to examine a dataset of ramen ratings and explore various trends within that dataset.

This dataset is called Ramen Ratings downloaded from Kaggle.

Variables Definition
review # Review number of the ramen (2580 reviews)
brand What brand the ramen is from
variety What the ramens name is, usually describing what flavor the noodles/seasoning are
style The packaging type of the ramen
country What country the ramen is from
stars How many stars the ramen received (1-5)

First, I load my packages.

library(dplyr)
library(stringr)
library(ggplot2)

Next, I am going to rename the original dataset and variables, so its easier to type.

# rename dataset
df <- ramen_ratings

# remove original dataset
rm(ramen_ratings)

df <- dplyr::rename(df,
                    "brand" = "Brand",
                    "variety" = "Variety",
                    "style" = "Style",
                    "country" = "Country",
                    "stars" = "Stars")

I am going to change some variables within the dataset to catagorize it properly and account for any missing or repeated information.

# change all "United States" to "USA"
df <- mutate(df, country = str_replace_all(country, "United States", "USA"))

# code variables into factors
df$style <- factor(df$style)

df$brand <- factor(df$brand)

df$variety <- factor(df$variety)

df$country <- factor(df$country)

# change stars into numeric variable
df$stars <- as.numeric(df$stars)

I am going to begin my making my first graph. This graph which type of packaging style is rated the most.

With this we can see that ramen with the Pack Packaging type appears the most. I also want to make another graph that shows what the average rating for each type of ramen packaging is.

I am first going to summarize the data by creating a new variable which groups all the packaging types and calculates the average amount of stars for each type. I also make sure to eliminate any null values.

summary_data <- df %>%
  group_by(style) %>%
  summarize(avg_rating = mean(stars, na.rm = TRUE))

With this summarized data, I am going to now create the graph with the packaging style and the average stars rated.

ggplot(summary_data) +
  geom_bar(aes(x= style, y = avg_rating), stat = "identity", fill = "purple", color = "black") +
  labs(title = "Average Rating for Types of Ramen Packaging",
       x = "Packaging Type",
       y = "Average Rating in Stars")

It seems that bar and box have the highest average ratings, however they have the least amount of ratings as seen in the previous graph. Pack has the third highest rating, but it appears the most frequently.

The next graph I will create will look at the countries that these ratings are based on.

top_countries <- df %>%
  count(country) %>%
  arrange(desc(n)) %>%
  slice_max(order_by = n, n = 10)

There are 37 countries that rated ramen in this dataset, but for easier and more concise graphs, I will only be including the top ten for each catagory that we will be graphing. In the code above, I have created a new variable called top_countries that arranges and selects the top ten countries with the highest frequencies of ratings.

ggplot(top_countries) +
  geom_bar(aes(x = country, y = n), stat = "identity", fill = "purple", color = "black") +
  labs(title = "Ten Countries with the Highest Frequency of Ratings",
       x = "Countries",
       y = "Frequency")

We can see Japan has the highest frequency of ratings followed by the USA. Next, I am going to create a graph to see the average ratings of the top ten countries. I will do this by creating a new variable that is called summary_country, and it will find the average ratings of the stars for each country and then display the top ten with the highest.

summary_country <- df %>%
  group_by(country) %>%
  summarize(avg_rating = mean(stars, na.rm = TRUE)) %>%
  top_n(10, avg_rating)

Then, I create a bar graph based on summary_country that I have created above.

ggplot(summary_country) +
  geom_bar(aes(x = country, y = avg_rating), stat = "identity", fill = "purple", color = "black") +
  labs(title = "Ten Countries with the Highest Average Stars",
       x = "Countries",
       y = "Stars")

In this graph, Brazil and Sarawak are the countries with the highest average rating, coming in at around 4.5 stars. However, it should be noted that these two countries were not on the list of countries with the highest frequency of ratings. Therefore, while these countries may have the highest stars, it may not be reflective of its quality.

After creating a few graphs, I will create a graph with the variables we have explored: stars, style, and country.

avg_ratings <- df %>%
  group_by(country, style) %>%
  summarize(avg_rating = mean(stars, na.rm = TRUE))
`summarise()` has grouped output by 'country'. You can override using the `.groups` argument.

I group the average ratings by country and style because those are the variables I want to include on the graph. After that, I make a bar graph that includes these variables and facet wrap country, so it will create a graph for each country.

ggplot(avg_ratings, aes(x = style, y = avg_rating, fill = style)) +
  geom_bar(stat = "identity") +
  facet_wrap(~ country) +
  labs(title = "Average Ramen Ratings by Packaging Type and Country",
       x = "Packaging Type",
       y = "Average Rating")

We can see in the graph that Japan and the US have the most variety of ramen packaging types, and they boast some of the highest ratings, as well as Malayasia. The pack type ramen seems to be the most prevalent with every country having the pack type. In Japan and Malaysia the box type ramen is the most highly rated, while in the U.S. it is the bar type ramen. However, the bar type ramen only appears in the U.S. graph. What is interesting, in fact, if we compare these to the previous graph, for example the countries with the highest average rating (Brazil and Sarawak), those countries only have one type of ramen, so the data is not showing the entire picture.

Conclusion:

We have created multiple graphs based on various variables, and we can take a look at the popularity of each ramen. The box and the bar are the most highly rated types, but they are also the types that appear the least. The pack has a moderately high rating, and it appears the most. The pack seems to be the most widespread packaging type and easy to purchase. For countries, Japan and the USA have the most ratings, but they are not the ones with the highest average star rating. However, Japan is among the top 10 with the highest average star rating, while the U.S., so it seems like Japan is the better choice.
